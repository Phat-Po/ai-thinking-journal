# 2026-05-14 Thursday

## 今日主題  
今天大部分時間都花在搞自動化每日思考摘要的 pipeline，還有修復 VIBE Dashboard 的健康檢查和啟動問題，順便處理了外接硬碟斷線導致的專案無法讀取狀況。

## 關鍵決策  
- 決定用本地 Ollama 服務跑每日摘要，因為 Anthropic API 太貴又不方便離線。  
- 針對 VIBE Dashboard，改用持續重試的健康檢查機制，避免後端掛掉後前端一直顯示離線不恢復。  
- 修正 VIBE Dashboard 的啟動腳本，從硬編碼路徑改成動態路徑，並用 AppleScript applet 替代純 bash 腳本，解決無法啟動問題。  
- Codex CLI 的 hooks 功能因版本不符被迫關閉，改手動移除 `--enable hooks` 參數來繞過錯誤。  
- 針對每日摘要，改成先對每個 session 做簡短預摘要，再合併成整天的總結，減少輸入大小並提升品質。  
- 決定先不做 Codex 歷史資料批量回填，先專注當天資料的提取和格式兼容。
